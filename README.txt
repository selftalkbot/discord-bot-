Private AI Discord Bot – User Guide

This bot creates a private channel for you inside the server and allows you to talk to an AI in that channel.

To create your private channel, type:
!private
The bot will create a text channel named 📝-yourusername. Only you, the bot, and server administrators can see it.

To use the AI, simply type messages inside your private channel. The AI will respond automatically.

If you see the message “AI monthly limit reached. AI has been temporarily disabled.” it means the free AI limit has been reached. The bot owner must restart the bot after the limit resets. Until then, the AI will not respond.

If the bot does not respond or your channel is not created, contact the server administrator.

The AI currently replies to each message individually and does not store long-term memory.

Use this bot for notes, brainstorming, study help, or private conversations with AI.
